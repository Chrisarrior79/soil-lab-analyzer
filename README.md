# Soil Lab Analyzer

Web application developed in Python and Flask to perform geotechnical engineering calculations.
Live app:
https://soil-lab-analyzer.onrender.com/

## Features
- Primary consolidation settlement calculation (Terzaghi theory, normally consolidated soils)

## Technologies
- Python
- Flask
- HTML
- Git
- Gunicorn
- Render (cloud deployment)

## Purpose
This project was created to combine geotechnical engineering and software development, providing practical engineering tools through a web interface.




Technologies used:
- Python
- Flask
